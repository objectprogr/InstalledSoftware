# Installed software

Simple script writing in powershell which get all installed software on your computer (Windows OS)

## Getting Started

Run script:
1. Clone script
2. Run powershell
3. Go to script location
4. Command execution:

```
.\InstalledSoftware.ps1
```

## Built With

Powershell version 5
