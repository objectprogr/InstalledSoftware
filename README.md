# Installed software

Simple script writing in powershell which get all installed software on your computer (Windows OS)

## Getting Started

Run script:
1. Download release
2. Extract files
3. Run powershell
4. Go to script location
5. Command execution:

```
.\InstalledSoftware.ps1
```

## Built With

Powershell version 5
